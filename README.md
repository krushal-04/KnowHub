# Knowledge Share Portal

## 📌 Project Overview

The **Knowledge Share Portal** is a web-based platform built using the Django framework that allows users to upload, share, and access educational resources in an organized way. The system enables students or users to browse categories, upload documents, and manage their shared resources efficiently.

---

## 🚀 Features

* User Registration and Login
* Upload Educational Documents
* Browse Documents by Category
* View Document Details
* Edit or Delete Uploaded Documents
* User Profile Management
* Public User Profiles
* Notifications System
* Search and Explore Learning Resources

---

## 🛠 Technologies Used

* **Backend:** Python, Django
* **Frontend:** HTML, CSS
* **Database:** SQLite
* **Version Control:** Git
* **Platform:** GitHub

## ⚙️ Installation Guide

### 1. Clone the Repository

git clone https://github.com/yourusername/knowledge-share-portal.git

### 2. Navigate to the Project Directory

cd knowledge-share-portal

### 3. Create Virtual Environment

python -m venv venv

### 4. Activate Virtual Environment

Windows:
venv\Scripts\activate

### 5. Install Required Dependencies

pip install -r requirements.txt

### 6. Apply Migrations

python manage.py migrate

### 7. Run Development Server

python manage.py runserver
